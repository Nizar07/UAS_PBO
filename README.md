# Aplikasi Manajemen Bengkel

Aplikasi Android yang dikembangkan untuk membantu proses administrasi bengkel dengan mengelola data pelanggan dan kendaraan secara lebih efisien. Aplikasi ini memudahkan petugas dalam mencatat pelanggan yang datang serta memberikan informasi kepada pelanggan ketika kendaraan telah selesai dikerjakan.

Fitur utama aplikasi meliputi:

- 🚗 Pencatatan data pelanggan dan kendaraan
- 📱 Penyimpanan informasi seperti nama pelanggan, nomor telepon, dan nomor polisi kendaraan
- 🛠️ Manajemen daftar kendaraan yang sedang dikerjakan
- 💬 Pengiriman notifikasi melalui **WhatsApp** secara otomatis ketika kendaraan telah selesai dikerjakan
- 📋 Antarmuka yang sederhana dan mudah digunakan untuk proses administrasi bengkel

Aplikasi ini dikembangkan sebagai proyek perkuliahan oleh **tim yang terdiri dari 2 orang**, dengan tujuan menerapkan konsep pengembangan aplikasi mobile untuk mendukung digitalisasi layanan bengkel.
